# Contributing

Thanks for your interest in contributing to Network Speed Monitor!

## Development Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start development mode
4. Make your changes
5. Test thoroughly

## Testing

- Test with different network interfaces (WiFi, Ethernet, VPN)
- Test network interface switching
- Test counter reset scenarios
- Verify speed calculations are accurate

## Pull Requests

- Keep PRs focused on a single feature or fix
- Update CHANGELOG.md with your changes
- Add tests if applicable
- Ensure `npm run lint` passes

## Issues

When reporting issues, please include:
- macOS version
- Raycast version
- Network interface type (WiFi, Ethernet, VPN, etc.)
- Steps to reproduce
- Expected vs actual behavior
