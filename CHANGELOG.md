### Added

- Real-time network speed monitoring in menu bar
- Dynamic network interface detection (supports WiFi, Ethernet, VPN, USB ethernet)

- Download and upload speed display with auto-formatting (B/s, KB/s, MB/s, GB/s)
- Menu bar integration with 1-second update interval
- Quick access to macOS Network Settings
- Persistent speed tracking across component remounts using Raycast Cache API
